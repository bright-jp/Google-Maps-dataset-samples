# Google Maps-dataset-samples

<h2>1001件のGoogle Mapsビジネスのサンプルデータセット</h2>

![Google Maps dataset header](https://github.com/bright-jp/Google-Maps-dataset-samples/blob/main/google-maps-datasets.PNG)

1000件を超えるビジネスのGoogle Mapsデータセットサンプルです。データセットは <b>Bright Data API</b> を使用して抽出されました。

<h2>この無料データセットに含まれるデータポイント:</h2>

* ```place_id```: Google Maps上の場所を識別する一意のIDです
* ```name```: ビジネスの名称です
* ```category```: ビジネスのカテゴリまたは種類です
* ```address```: ビジネスの所在地住所です
* ```lat```: ビジネス所在地の緯度座標です
* ```lon```: ビジネス所在地の経度座標です
* ```url```: Google Maps上のビジネスを指すURLです
* ```country```: ビジネスが所在する国です
* ```phone_number```: ビジネスの連絡先電話番号です
* ```main_image```: ビジネスに関連付けられたメイン画像のURLまたは識別子です
* ```reviews_count```: ビジネスのレビュー総数です
* ```rating```: 顧客による平均評価です
* ```reviews```: 顧客が残したレビューまたはコメントです
* ```open_hours```: ビジネスの営業時間です
* ```open_hours_updated```: 最終更新された営業時間です
* ```price_range```: ビジネスに関連する価格帯です
* ```services_provided```: ビジネスが提供するサービスの一覧です
* ```open_website```: 訪問可能なWebサイトをビジネスが持っているかどうかのリンクまたは表示です
* ```category_search_input```: この結果を取得するために使用されたカテゴリ検索入力です

ほかにも多数あります。

これは、「Google Maps Businesses (public data)」から派生したサンプルサブセットです。
当該データセットには <b>46,000,000件を超えるビジネス</b> が含まれています。

利用可能なデータセットのファイル形式: <b>JSON, NDJSON, JSON Lines, CSV, または Parquet。必要に応じて、ファイルは .gz に圧縮できます</b>。

データセット配信タイプのオプション: <b>Email, API download, Webhook, Amazon S3, Google Cloud storage, Google Cloud PubSub, Microsoft Azure, Snowflake, SFTP</b>。

更新頻度: <b>Once, Daily, Weekly, Monthly, Quarterly, または Custom basis</b>。

抽出されたデータポイントに追加できるデータエンリッチメント: <b>Based on request.</b>

<b>[Google Mapsのフルデータセットを取得する](https://brightdata.jp/products/datasets/google-maps)</b>。

<h2>Google Mapsデータセットのユースケースは何ですか？</h2>

<h3>1. 競合分析</h3>
Google Mapsデータセットを活用して、特定地域における競合を特定し分析します。レビューと評価データにより、企業は競合の強みと弱みについての洞察を得られ、people & jobsデータセットはCRM戦略の強化にも役立ちます。

<h3>2. ロケーションベースのマーケティング</h3>
Google Mapsデータセットは、企業がマーケティング戦略を精緻化するための広範な位置情報データを提供します。ユーザー行動を分析し、人気の目的地を特定することで、企業はジオターゲティング型マーケティングキャンペーンに向けて取り組みを最適化できます。

<h3>3. 市場戦略</h3>
Google Mapsデータセットを活用して、高度にターゲットを絞ったマーケティング戦略や営業アウトリーチ計画を策定します。このデータには包括的なビジネス詳細、評価、レビューが含まれており、企業は世界規模で主要な見込み客や競合を特定できます。
